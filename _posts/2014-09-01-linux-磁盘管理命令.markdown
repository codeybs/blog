-------
`cd`

+ 功能描述：切换目录.
+ 简单使用描述：
    + `cd （无参数）`:切换到环境变量HOME所指的目录下，通常为用户的主目录. 
    + `cd ~` ：切换到主目录.
    + `cd - `：返回上次切换前的目录.
    + `cd dirname` : 切换到名为dirname(`可以是绝对路径或者相对路径`)的目录下.
        + `.` 表示当前目录.
        + `..`表示上层目录.  
        
-------

`df`

+ 功能描述：报告文件系统磁盘使用状况.
+ 简单使用描述：
    + `-a,--all`: 显示所有，包括虚拟文件系统.
    + `-h, --human-readable`:以可读的形式打印信息.
    + `-i, --inodes` : 列出inode信息而非块使用量信息.

-------

`dirs`

+ 功能描述：显示目录记录.
+ 简单使用描述：`dirs [-clpv] [+N] [-N]`
