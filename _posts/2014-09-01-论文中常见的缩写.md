## 论文中常见的一些英文缩写

+ LSH ：left side hand.
+ RSH : right side hand.
+ e.g. : exempli gratia(拉丁文), =for example.
+ etc : etcetera,= and so on.
+ i.e. :id est(拉丁文), =that is 
+ w/o : without.
+ w.r.t : with regard to.
+ w.l.o.g : without loss of generality.
+ iff. : if and only if.
+ r.v. : random variable.
+ s.t. : subject to.
